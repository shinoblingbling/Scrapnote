Scrapnote Privacy Policy

Last updated: 2026-04-22

Scrapnote ("the Extension") is a Chrome extension that classifies web articles using AI and saves them to Notion. This privacy policy explains what data is collected, how it is used, and how it is protected.

1. Data We Collect

When you activate the Extension (via shortcut or popup button), the following data is collected from the currently active tab only:

DataPurpose
Page titleSaved as the article title in Notion
Page URLSaved as a reference link in Notion
Page content (text)Sent to AI for classification; optionally saved to Notion
Author name, site nameSaved as metadata in Notion

The Extension does not collect data from any page other than the one you explicitly activate it on. It does not track browsing history, collect personal information, or run in the background.

2. Third-Party Services

The collected data is sent to the following third-party services:

ServiceData sentPurposeRetention
Groq API Title, URL, first 4,000 characters of content AI-based article classification (tags, category, summary) Not retained after processing (per Groq's policy)
Notion API Title, URL, full content, classification results Saving the article to your Notion database Stored in your Notion workspace until you delete it

3. API Keys

The Extension requires you to provide your own API keys (Groq API Key, Notion Integration Token). These keys are:

  • Stored locally on your device using chrome.storage.local
  • Encrypted with AES-256-GCM before storage
  • Never sent to any server other than the respective API endpoints (Groq, Notion)
  • Never collected by the Extension developer

4. Data Storage

  • All Extension settings (API keys, tag seeds, preferences) are stored locally in your browser using chrome.storage.local
  • No data is stored on any server operated by the Extension developer
  • Uninstalling the Extension removes all locally stored data

5. Data We Do NOT Collect

  • Browsing history or activity on pages where the Extension is not activated
  • Personal information (name, email, address, etc.)
  • Analytics, telemetry, or usage statistics
  • Cookies or tracking identifiers

6. User Control

  • The Extension only activates when you explicitly press the shortcut (Ctrl+Shift+S) or click the popup button
  • You can delete saved articles directly from your Notion workspace
  • You can revoke API keys at any time from Groq Console and Notion settings
  • Uninstalling the Extension removes all local data

7. Security

  • API keys are encrypted with AES-256-GCM before local storage
  • All communication with third-party APIs uses HTTPS
  • The Extension does not inject scripts into pages except when explicitly activated
  • Source code is open source and available on GitHub

8. Children's Privacy

The Extension is not directed at children under 13 and does not knowingly collect data from children.

9. Changes to This Policy

This privacy policy may be updated when new features are added. Changes will be reflected on this page with an updated date.

10. Contact

If you have questions about this privacy policy, please open an issue on the GitHub repository.
