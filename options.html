Scrapnote 設定

初回のみAPIキー等を設定してください。その後は Ctrl+Shift+S で保存できます。

⚠ 注意

この拡張機能は、閲覧中のページの内容(タイトル・URL・本文)を Groq API と Notion API に送信します。
社内ポータル、メール、金融サイトなど機密性の高いページでは使用しないでください。

🔑 API認証情報

Groq Console で発行してください(無料、課金設定不要)
notion.so/profile/integrations で「+ New integration」から発行してください
Notionデータベースを開き、URLの「notion.so/{workspace}/ここの32文字?v=...」の部分。
データベースの「…」メニュー → 「コネクト」 → 作成した Integration を接続してください。

📋 Notionデータベースに必要なプロパティ

以下のプロパティ名・型でNotionデータベースを作成してください。プロパティ名は大文字小文字も含めて完全一致が必要です。
プロパティ名 型 用途
Titleタイトル記事タイトル(デフォルトの「名前」列をリネーム)
URLURL記事URL
TagsマルチセレクトAI生成タグ(最大5つ)
CategoryセレクトAI生成カテゴリ
PriorityセレクトHigh / Mid / Low
StatusセレクトUnread / Reading / Done / Archived
SummaryテキストAI生成要約
Read Time数値読了予測分数
Added日付保存日時(自動設定)
作成後、データベース右上の「...」メニュー → 「コネクト」から作成済みの Integration を接続してください。

🏷️ タグ初期シード(任意)

AIは Notion DB の既存タグを自動学習し、優先的に再利用します。
ここにタグを設定すると初期候補として使われます。空にしてもAIが記事内容から自動生成します。

📁 カテゴリ

AI分類時に1つだけ選ばせる大分類。JSON配列で編集可能。

⚙️ その他

✅ 保存しました

❓ トラブルシューティング

保存に失敗した場合、エラーメッセージを確認して以下を参考にしてください。

Notionへの保存に失敗しました (400)

  • "XXX is not a property that exists" — Notionデータベースに対象プロパティがありません。Title, URL, Tags, Category, Priority, Status, Summary, Read Time, Added がすべて存在するか確認してください
  • "is expected to be type Y" — プロパティの型が違います。READMEのプロパティ表を確認してください

Notionへの保存に失敗しました (404)

  • Database IDが間違っているか、Integrationがデータベースに接続されていません
  • データベースの「...」メニュー → 「コネクト」から Integration を接続してください

Notionへの保存に失敗しました (401)

  • Notion Integration Tokenが不正です。ntn_で始まるキーを再発行してください

AI分類に失敗しました (401)

  • Groq API Keyが不正です。gsk_で始まるキーを Groq Console で再発行してください

AI分類に失敗しました (429)

  • Groq APIのレート制限に達しました(30 RPM / 14,400 RPD)。少し待ってから再試行してください

特定サイトで本文抽出が失敗する

  • JavaScriptが多用されたSPA(X等)では本文取得に失敗することがあります。タイトルとURLのみ保存されます
  • chrome://、Chrome Web Store、新規タブでは仕様上動作しません
